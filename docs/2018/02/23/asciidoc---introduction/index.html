AsciiDoc - Introduction

2018-02-23 · 321 words · 2 minute read
Language
ArgParse · Intro

Introduction

AsciiDoc is a markup language (similar to Markdown).
AsciiDoctor is a tool to generate html (and PDF) from it.

Documents have following extension: .adoc

Official documentation can be found in http://asciidoctor.org/docs/user-manual

In the command lines listed below ,the $ corresponds to the prompt: it shall not be typed when entering the command.

Some nice slides to introduce are available here (done

Installation

More details in http://asciidoctor.org/#installation

  • Install Ruby

  • Install asciidoctor

$ gem install asciidoctor

  • Install a renderer: pygments, coderay, or rouge [for PDF only]

$ gem install pygments.rb
$ gem install coderay
$ gem install rouge

  • Install asciidoctor-pdf if you want to generate PDF

$ gem install asciidoctor-pdf --pre

You must have Python 2 installed to use pygments renderer: Python 3 is not working.

You should have python2 in your path.
(if python refers to Python 3 it is fine though).

Usage

Possible Outputs

  • HTML

  • PDF (asciidoctor-pdf)

  • Slides (by using backends)

    • DZSlides

    • Revealjs

    • Deck.js

    • Bespoke.js

HTML

To generate html:

$ asciidoctor <file>.adoc
$ asciidoctor -D <outdir> <file>.adoc
$ asciidoctor -d <outputfile> <file>.adoc

To have an incremental number for the sections & subsections:

$ asciidoctor -n <file>.adoc

PDF

To generate PDF (after installation of asciidoctor-pdf):

$ asciidoctor-pdf <file>.adoc

Slides

Documentation for Reveal.js and Deck.js are here:

  • Generating slides with Reveal.js

  • Install and use Deck.js backend

In most cases, you need the backends (https://github.com/asciidoctor/asciidoctor-backends):

  • DZSlides

  • Deck.js

  • Reveal.js

  • Bespek.js

An alternate version of backend for Reveal.js has been desribed in this blog.

Syntax

A summary of the syntax can be found here: http://asciidoctor.org/docs/asciidoc-syntax-quick-reference

Example.adoc
Hello here. I like *bold* and _italics_.

This is some text
on a single line.

This is some text ... +
on 2 lines.

Here is some code:
[source, python]
print("Hello MP")

And this is how it will look like:

Hello here. I like bold and italics.

This is some text on a single line.

This is some text …​
on 2 lines.

Here is some code:

print("Hello MP")
